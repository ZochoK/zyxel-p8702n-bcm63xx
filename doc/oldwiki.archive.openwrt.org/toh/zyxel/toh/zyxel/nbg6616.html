Upgrade from ZyXEL customized OpenWrt

On the ZyXEL firmware there is one large mtd partition for the kernel and everything. SSH into the router, user “root” and password (default is “1234”), and write the sysupgrade image to /dev/mtd6: 

# mtd -r write /tmp/firmware.bin /dev/mtd6

The following is a more detailed example of installing OpenWrt. 

ssh root@192.168.1.254
cd /tmp/
# the prompt should show root@nbg6161:/tmp
wget http://downloads.openwrt.org/releases/18.06.2/targets/ar71xx/generic/openwrt-18.06.2-ar71xx-generic-NBG6616-squashfs-sysupgrade.bin -O firmware.bin
mtd -r write /tmp/firmware.bin /dev/mtd6
Unlocking /dev/mtd6 ...
Writing from /tmp/firmware.bin to /dev/mtd6 ...     
Rebooting ... [w]

Now you can configure the device via http://192.168.1.1/

Recovery

The device is able to flash from a tftp server. You can also abort the boot loader via serial if a cable is attached to a LAN port.

  • Configure your server to 192.168.1.33, netmask 255.255.255.0
  • Attach to one of the LAN ports
  • Copy your firmware file into the tftp directory and name it ras.bin (IE download it from zyxel directly)
  • Turn on device holding down WPS key
  • Keep holding WPS key until the firmware is downloaded and router is flashing
  • Power cycle device when Power and USB2 LEDs flash. (It will take more than 5 minutes)

Technical note: header

Building an custom image that can be flashed currently requires adding a header. Either use genImgHdr from the open source bundle or implement a new header writer for OpenWrt (documentation is in the Makefile).

How to connect to the Serial Port of this specific device:

The device uses TTL @ 3.3V and not a standard RS-232 Serial that operates between 3 and 15V, so do not try to connect it to a common serial adapter: you will certainly fry the serial circuit or even the whole board. There are plenty of USB to TTL and RS-232 to TTL available on the market, just be careful with the voltage: the standard is 5V and it may also damage your board. Look for the ones with 3.3V or with both voltages and a way to switch between them.
