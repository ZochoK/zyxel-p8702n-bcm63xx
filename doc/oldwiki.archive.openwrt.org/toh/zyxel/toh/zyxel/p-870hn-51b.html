All these versions have significant bugs:
1) switch - only two middle LAN ports (labelled 2 and 3) work
2) wireless - bcm43222 isn't recognized
3) LED - LED state does not reflect the state of the router

SMP-enabled build is required for dual-core support

ADSL driver on orginal fw - A2pv6C033a.d23e

The Board ID is the same as the Netgear DGND3700 v1
