→ Install OpenWrt (generic explanation)

OpenWrt can be installed either through the OEM firmware web interface or with the use of the U-boot TFTP recovery functionality. Having a serial console is recommended for the latter method as it allows to monitor the router status while it is being flashed.

Flash Layout

Layer0 raw flash 8192 KiB
Layer1 mtd0 u-boot 192 KiB mtd1 u-boot-env 64 KiB mtd2 factory 64 KiB mtd3 firmware 7872 KiB
Layer2 mtd4 kernel 1403,61 KiB mtd5 rootfs 6468,39 KiB
Layer3 2412,39 KiB mtd6 rootfs_data 4056 KiB

OEM easy installation

  • Use a PC to browse to http://my.keenetic.net.
  • Go to the System section and open the Files tab.
  • Under the Files tab, there will be a list of system files. Click on the Firmware file.
  • When a modal window appears, click on the Choose File button and upload the firmware image.
  • Wait for the router to flash and reboot.
  • Telnet to 192.168.1.1 and set a root password or browse to http://192.168.1.1 if LuCI is installed.

OEM installation using the TFTP method

  • Download the latest firmware image and rename it to komni_recovery.bin.
  • Set up a Tftp server on a PC (e.g. Tftpd32) and place the firmware image to the root directory of the server.
  • Power off the router and use a twisted pair cable to connect the PC to any of the router's LAN ports.
  • Configure the network adapter of the PC to use IP address 192.168.1.2 and subnet mask 255.255.255.0.
  • Power up the router while holding the reset button pressed.
  • Wait approximately for 5 seconds and then release the reset button.
  • The router should download the firmware via TFTP and complete flashing in a few minutes.
  • After flashing is complete, use the PC to browse to http://192.168.1.1 and proceed with the configuration.

Specific values needed for tftp

Required Tftp server IP address 192.168.1.2
Firmware image Firmware image must be renamed to komni_recovery.bin

Opening the case

Note: This will void your warranty!

Turn the router upside-down and unstick two round rubber caps that cover screws. Use a cross-head screwdriver to remove the two screws. Then turn the router over back to its normal position and remove the top plastic cover. Unfortunately, there seems to be no way to do this without damaging at least some of the fasteners. However, due to the design of the case broken fasteners do not cause any trouble after reassembly.
