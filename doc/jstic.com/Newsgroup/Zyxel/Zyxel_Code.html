Actualización del Firmware del Prestige 650HW-31(E)

Archivos Necesarios:

  • Firmware Original Telefónica 3.40KM2 Con el que vienen los routers de Telefónica.
  • Firmware Nuevo ZyXEL 3.40KW4b1
    • Configuración Web: http://10.0.0.1:2033 Login:admin Password:1234
  • Bootbase 1.11 necesario para actualizar el firmware
    • Descargar archivo ZIP - Descargar archivo sin comprimir
  • Configuración Firmware KW4b1 lista para funcionar con Telefónica 
    • Multipuesto con IP dinámico
    • Dirección IP: 192.168.1.1
    • Configuración Web: Login: adslppp@telefonicanetpa Password: adslppp
    • Configuración Telnet: Password: adslppp
    • Wireless habilitado, sin encriptación WEP ni filtrado de MAC
    • Administración sólo posible desde LAN
    • DHCP deshabilitado
    • Filtros y tabla NAT por defecto
    • Datos completos de configuración
  • Firmware ZyXEL V3.40(IS.2)C0  -  Compatible con ADSL2+ y ADSL2 pero deshabilita la función Wireless
  • Firmware ZyXEL v3.40 KW5d0 - Versión mejorada del firmware. Soporta WiFi, pero no soporta ADSL2/2+. Actualización recomendada. (Gracias Marco por la confirmación)

Misceláneos

  • Esquema cable consola. Para quienes perdísteis el cable o no lo tengáis disponible, aquí está el mapa para construirlo vosotros mismos. Muy sencillo.

Actualización del Firmware del Prestige 660HW-61

Archivos Necesarios:

  • Firmware Nuevo Telefónica 3.40PT0b46 - Última versión del firmware de Telefónica

  • Firmware Nuevo ZyXEL 3.40PE.9C0 - Firmware original ZyXEL estable

  • Firmware Nuevo ZyXEL 3.40PE.10C0 - Firmware original ZyXEL INESTABLE - Presenta problemas de estabilidad WiFi

  • Firmware Nuevo ZyXEL 3.40PE.11C0 - Firmware original ZyXEL. Según algunas fuentes el WiFi es más estable. Particularmente lo veo tan inestable como la anterior. Descarga externa directa de ZyXEL

Misceláneos

  • Bootbase 1.10t - Bootbase con la que vienen los routers de Telefónica

    • ADVERTENCIA: En el Bootbase se guarda la dirección MAC del router. Esta Bootbase fue extraída de un ZyXEL de Telefónica y la MAC fue modificada a FF FF FF en la posición de 3FF0. Por lo tanto, tenéis que buscar un editor hexadecimal y editarlo con la vuestra. Si no lo hacéis correctamente PODEIS CREAR CONFLICTOS DE MAC
  • Bootbase 1.06 - No disponible por los momentos. Si alguien la tiene, por favor que me la facilite.
  • Esquema cable consola. Este cable es mucho más complicado de hacer, pero es necesario para actualizar.
